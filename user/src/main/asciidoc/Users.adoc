[cols="h,5a"]

| GET

| Roles allowed

|

! No

\| _birthDate_ \| _ phoneNumber_ \| _preferredLocale_ \| _creationDate_ \| _updateDate

! isDesc
! No
! sort direction. Defaults to _false_. +

_true_ \| _false_
